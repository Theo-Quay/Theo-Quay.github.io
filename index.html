Projects

Satellite imagery unsupervised segmentation mapping - 2024

In this project, an unsupervised learning model developed by Saha et al, 2022, was used to create segmentation maps of Earth sentinel-2 satellite imagery. The goal of this project was to create a python pipeline integrating the process of requesting and downloading satellite imagery from the sentinelhub API, with the segmentation of said images using an unsupervised learning program developed previously. While Saha's model was orignally developed with RGB imagery in mind, my code allows for easier investigation on how changing the sentinel-1 & 2 bands the model runs on can affect how the image in segmented. This project was done in collaboration with W. Hamish Mitchell (whamitch@mit.edu) and Gaia Stucky de Quay (gaias@mit.edu) through MIT.

Technologies: Python, Machine Learning, APIs

Research

Vacancy-ordered Double Perovskites - 2023

My research at entailed the analysis of x-ray degradation of lead-free halide perovskites with applications in optoelectronics using various analytical techniques. The aim of this research was to expand on Robert Palgrave’s group’s knowledge on halide perovskites, specifically A2BX6 vacancy ordered double perovskites, under the guidance of Prajna Bhatt (PhD student) and Anna Regoutz (Professor, placement supervisor). In my eight weeks at the UCL Chemistry Department, I was able to:

  • synthesise and obtain brand-new crystallographic data for triclinic FA2SnBr6 using PXRD;
  • b) synthesise cubic-phase (NH4)2SnBr6;
  • c) create a phase series between both crystals by doping FA (FA = HCN2H4+) into the (NH4)2SnBr6 structure;
  • d) use XPS to measure the analytical doping percentages of the aforementioned series;
  • e) measure direct and indirect band gaps for the substances synthesised;
  • f) collect x-ray radiation damage data on certain samples.
  • Overall, this research opportunity significantly developed my skills as an inorganic and analytical chemist. It expanded my knowledge on crystallography, band gap theory, analytical techniques and more.
  • This research was made possible by the ACTF undergraduate research bursary.
